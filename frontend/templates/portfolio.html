Placement Portal

The Navion Placement Portal is a web-based application developed to manage and organize campus placement activities efficiently. It allows the placement administrator to add, update, view, and delete company details such as company name, role, location, and salary package. Student information for upcoming placements can be collected through Google Forms and stored securely in the database. The system is developed using Python Django and uses MySQL for structured data storage and management. This project reduces manual work and provides a centralized platform for placement data handling.

Number Guessing Game

It is a simple mini project where the computer randomly selects a number within a given range (for example 1 to 100), and the player has to guess the correct number. After each guess, the program gives hints such as “Too High” or “Too Low” to help the player reach the correct answer. The game continues until the user guesses the exact number. It can also include a feature to count the number of attempts taken by the player. This project helps in understanding concepts like loops, conditional statements, random module, and user input handling in Python.
